Para crear el indice

PUT /zapatillas_deportivas
{
  "mappings": {
    "properties": {
      "name": { "type": "text" },
      "price": {
        "type": "scaled_float",
        "scaling_factor": 100
      },
      "brand": { "type" : "text" },
      "description": { "type" : "text" }
    }
  }
}

Para agregar mas propiedades al documento
PUT /zapatillas_deportivas/_mapping
{
  "properties": {
    "color":  { "type": "text"},
    "modelo":  { "type": "text"}
  }
}
